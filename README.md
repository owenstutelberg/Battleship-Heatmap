# Battleship-Heatmap
Simple Battleship board game solver.

# Usage
- clicking on the names of the ships will change their "existence" (when you sink a ship, make sure to click the name so that the code recognizes that the ship is sunk).
- best squares will be highlighted in green.

- clear button will clear the selection grid.

- board starts as an empty ocean

- first click will mark a hit
- second click will mark a hit and sink
- third click will mark a miss
- fourth click will mark it empty again
